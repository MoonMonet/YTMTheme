# Contributing to MoonMonet YTM Theme

Thank you for wanting to contribute! Any help is appreciated to make this theme look and work better for everyone.

## ✨ Getting Started

1. Fork the repo.
2. Clone your fork:
	```bash
	git clone https://github.com/YourUsername/YTMTheme.git
	```
3. Make your changes in the relevant file in the `/components` or `/base` directories.
4. Test your changes locally by loading the modified CSS files in your browser.

## 🐛 Issues
If you find a bug or have a feature idea, please open an issue using the provided templates.

## 🛠️ Pull Requests
1. Create a new branch (`feature/AmazingFeature` or `fix/BugFix`).
2. Commit your changes.
3. Push to the branch.
4. Open a PR and describe what you changed!
