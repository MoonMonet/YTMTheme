MoonMonet YTM brings a purplish accent look to your music experience.

## ✨ Why MoonMonet YTM?

- **Material You Design:** Clean, modern, and purplish aesthetic following Material Design 3 guidelines.
- **Code Structure:** Separated into clean components for easier customization and maintenance.
- **Support:** Works on both browser (Stylus) and the standard desktop app (`th-ch`).

## 🚀 How to install

### Stylus (Browser)

1. Install the [Stylus extension](https://add0n.com/stylus.html) for your browser.
2. Click [this link to install](https://raw.githubusercontent.com/MoonMonet/YTMTheme/main/moonmonet.css).
3. Confirm the installation in Stylus.

### Desktop App (th-ch)

You can inject the CSS directly into the settings or use a plugin manager to load the remote URL.

## 🏗️ Structure

The theme is modularized for better organization:

- `base/`: Core variables and global resets.
- `components/`: Specific UI components like the player bar, buttons, and navigation.
- `pages/`: Page-specific overrides.
- `ui/`: Typography and scrollbar styling.
